## config parameter

| key                | default | possible values | comment                              |
|:-------------------|:--------|:----------------|:-------------------------------------|
| DriverEncoding     |         |                 | if not configured then UTF-8 is used |
| Decimals           | 2       |                 | 0 = amount remains the same          |
| RoomLength         | 4       |                 |                                      |
| WaitForReplyPacket | true    | true/false      |                                      |

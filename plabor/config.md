## config parameter

| key                       | default | possible values   | comment                                          |
|:--------------------------|:--------|:------------------|:-------------------------------------------------|
| DriverEncoding            |         | Windows-1252      | if not configured then UTF-8 is used             |
| DirectWakeupMode          | false   | true/false        |                                                  |
| WakeUpDateWithYear        | true    | true/false        | date as ddmmyyyy or ddmm                         |
| RemoveLeadingZeroes       | true    | true/false        |                                                  |
| IsProdac                  | false   | true/false        |                                                  |
| RoomNumberAsReservationID | false   | true/false        |                                                  |

## config parameter XmlPos

| key            | default | possible values | comment                                                        |
|:---------------|:--------|:----------------|:---------------------------------------------------------------|
| DriverEncoding |         | Windows-1252    | if not configured then UTF-8 is used                           |
| EntityEncoding | false   | true/false      | use xml compatible entity encoding, DriverEncoding must be set |
| Decimals       | 2       |                 | 0 = amount remains the same                                    |
